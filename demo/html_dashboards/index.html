About These Demos

These standalone HTML demos showcase the core concepts of Digital Signal Intelligence (DSI) without requiring any server or Python installation. Simply open any HTML file in your browser.


For a live demonstration that runs the actual DSI workflow engine, see the Live Demo Server which requires Python and FastAPI.


Key Concepts Demonstrated:

  • Signal extraction from external sources
  • Weighted scoring across signal groups
  • Tier assignment and override logic
  • Premium calculation with ILF and modifiers
  • Coverage-specific signal configurations
